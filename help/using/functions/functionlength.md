# length {#length}

Devuelve el número de caracteres de la expresión de cadena en el parámetro .

## Categoría

Cadena

## Sintaxis de función

`length(<parameters>)`

## Parámetro

* string

## Firma y tipo devuelto

`length(<string>)`

Devuelve un número entero.

## Ejemplo

`length("Hello World")`

Devuelve 11.
