# currentTimeInMillis {#currentTimeInMillis}

Devuelve la hora actual en milisegundos epoch.

## Categoría

Fecha

## Sintaxis de función

`currentTimeInMillis()`

## Parámetros

Esta función no utiliza parámetros.

## Firmas y tipo devuelto

`currentTimeInMillis()`

Devuelve un entero.

## Ejemplos

`currentTimeInMillis()`

Devuelve &quot;1544712617131&quot;.
