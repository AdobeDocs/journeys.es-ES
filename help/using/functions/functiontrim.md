# trim {#trim}

Elimina los espacios inicial y final.

## Categoría

Cadena

## Sintaxis de función

`trim(<parameters>)`

## Parámetro

| Parámetro | Tipo |
|-----------|------------------|
| string | string |

## Firma y tipo devuelto

`trim(<string>)`

Devuelve una cadena.

## Ejemplo

`trim(" Hello ")`

Devuelve &quot;Hello&quot;.
