# upper {#upper}

Devuelve una versión en mayúsculas del parámetro.

## Categoría

Cadena

## Sintaxis de función

`upper(<parameters>)`

## Firma y tipo devuelto

`upper(<string>)`

Devuelve una cadena.

## Ejemplo

`upper("b")`

Devuelve &quot;B&quot;.
