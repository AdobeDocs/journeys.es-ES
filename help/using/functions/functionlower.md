# lower {#lower}

Devuelve una versión en minúscula del parámetro.

## Categoría

Cadena

## Sintaxis de función

`lower(<parameter>)`

## Parámetro

* string

## Firma y tipo devuelto

`lower(<string>)`

Devuelve una cadena.

## Ejemplo

`lower("A")`

Devuelve &quot;a&quot;.
