# contain {#contain}

Comprueba si la segunda cadena de argumento está incluida en la primera cadena de argumento.

## Categoría

Cadena

## Sintaxis de función

`contain(<parameters>)`

## Parámetros

* string

## Firma y tipo devuelto

`contain(<string>,<string>)`

Devuelve un valor booleano.

## Ejemplo

`contain("rowing is great', "great")`

Devuelve true.
